  1. Construct the circle having diameter \(Z_0Z\text{.}\) Plot a point at which this circle intersects \(c\) and label this point \(T\text{.}\) Hint
  2. Construct the perpendicular to \(Z_0Z\) through \(T\text{.}\)
  3. Let \(Z^*\) be the intersection of this perpendicular with segment \(Z_0Z\text{.}\)
  4. Hide intermediate constructions so that only \(Z\) and \(Z^*\) remain. If you move \(Z\) back inside the circle, the “other” \(Z^*\) should still reappear. Check to make sure.
