Section 2.1 Getting to know Geogebra

You can access Geogebra apps, which are free to use, in a few ways. Click on the Calc button near the top of this page to open a Geogebra window on the right of this screen. This can be handy from time to time. You can also open the Geogebra apps at their website geogebra.org.

Lots of good Geogebra demonstrations are available in their gallery. We will use some demonstrations specific to this text as well. For class activities, it will be best to use Geogebra from their website rather than from the calculator embedded on this page, if only because the working window on your screen is likely bigger.

Activity

Worksheet Welcome to Geogebra

At geogebra.org look for the list of apps under the heading Powerful Math Apps. For this we want to use the app geometry.

When you open the Geometry app, you should see a blank canvas on the right and a pallette of tools on the left. We will want more than the basic tools, so click on MORE to see them all.

To familiarize yourself with this app, work through the following constructions.

1.

Construct a circle. Label the center of the circle \(Z_0\text{.}\) Label the circle itself as \(c\text{.}\) One quirk with Geogebra is that points must be labelled by capital letters, and lines and circles must be labelled by lower case letters.

To change an object's label, click the Move tool and select the object. A window should appear with a text box. Click that to change the label. To get \(Z_0\) type Z_0.

2. Construct \(z^*\) if \(z\) is inside the circle.

In this exercise we construct the point symmetric to \(Z\) with respect to the circle \(c\) where \(Z\) is inside the circle.

  1. Construct a point inside the circle, and label it \(Z\text{.}\) Then construct the ray \(\vec{Z_0Z}\text{.}\)
  2. Construct the line perpendicular to ray \(\vec{Z_0Z}\) at the point \(Z\text{.}\) Call this line \(m\) (lines get lower case letters).
  3. With the point tool selected, define one of the points at which \(m\) intersects the circle and label this point \(T\text{.}\)
  4. Construct segment \(Z_0T\) and then construct the line perpendicular to this segment at \(T\text{.}\) Call this line \(n\text{.}\)
  5. Define the point at which line \(n\) intersects ray \(\vec{Z_0Z}\text{.}\) This is the point symmetric to \(Z\) with respect to circle \(c\text{.}\) Label this point \(Z^*\text{.}\)
3. Hide intermediate constructions..

Select the Show/Hide Object tool and click on these intermediate constructions: ray \(\vec{Z_0Z}\text{,}\) lines \(m\) and \(n\text{,}\) the point \(T\text{,}\) and segment \(Z_0T\text{.}\) Now click on the Move tool (or another tool). Those objects (and hopefully nothing else) will disappear. To get them to reappear, click on the Show/Hide Object tool again. The objects marked for hiding will appear faiter than the other objects. Note: If you simply delete these objects all other objects depending on them (like \(Z^*\)) will vanish too and we don’t want that!!

Congratulations!! You constructed the point symmetric to \(Z\) with respect to \(c\text{.}\) Now click on the Move tool and drag the point \(Z\) around inside the circle. Admire how \(Z^*\) follows dutifully around and is always the point symmetric to \(Z\text{.}\) Notice how \(Z\) and \(Z^*\) get closer together as you near the circle, and how \(Z^*\) rushes excitedly toward infinity, like a dog wanting to please its master, as \(Z\) marches toward \(Z_0\text{.}\) What happens if \(Z\) moves outside the circle? Can you construct \(Z^*\) if \(Z\) is outside \(c\text{?}\)

4. Construct \(z^*\) if \(z\) is outside the circle..

Move \(Z\) outside the circle and complete these steps.

  1. Construct the circle having diameter \(Z_0Z\text{.}\) Plot a point at which this circle intersects \(c\) and label this point \(T\text{.}\) Hint
    Begin by finding the midpoint of segment \(Z_0Z\text{.}\)
  2. Construct the perpendicular to \(Z_0Z\) through \(T\text{.}\)
  3. Let \(Z^*\) be the intersection of this perpendicular with segment \(Z_0Z\text{.}\)
  4. Hide intermediate constructions so that only \(Z\) and \(Z^*\) remain. If you move \(Z\) back inside the circle, the “other” \(Z^*\) should still reappear. Check to make sure.
